# startupjs cli

> start, watch and build startupjs project

## Installation

```sh
yarn add @startupjs/cli
```

## Usage

```sh
npx startupjs init myapp && cd myapp
```

```sh
npx startupjs server
```

```sh
npx startupjs web
```

```sh
npx startupjs metro
```

```sh
npx startupjs android
```

```sh
npx startupjs ios
```
