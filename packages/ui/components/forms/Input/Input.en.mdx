# Input

Input provides a wrapper api around [Checkbox](/docs/en/components/Checkbox), [TextInput](/docs/en/components/TextInput) & [ObjectInput](/docs/en/components/ObjectInput) components by adding two-way data bindings. You can use any of the above components by specifying the `type` prop.

```jsx
import { Input } from from '@startupjs/ui'
```

## Example

```jsx example
const [, $textValue] = useValue()
const [, $checkboxValue] = useValue()
const [, $objectValue] = useValue()
return (
  <Div>
    <Input
      type='text'
      label='Text input'
      $value={$textValue}
    />
    <Br />
    <Input
      type='checkbox'
      label='Checkbox component'
      $value={$checkboxValue}
    />
    <Br />
    <Input
      type='object'
      $value={$objectValue}
      properties={{
        checkbox: {
          input: 'checkbox',
          label: 'Click to display next input'
        },
        dependent: {
          dependsOn: 'checkbox',
          dependsValue: true,
          input: 'text',
          label: 'Dependent on checkbox',
          style: { marginTop: u(1) }
        }
      }}
    />
  </Div>
)
```
