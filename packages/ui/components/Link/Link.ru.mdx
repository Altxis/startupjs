# Link (ссылка)

Ссылка позволяет перемещаться по приложению и настраивать якоря.

## Пример

```jsx
import { Link, Span } from '@startupjs/ui'
```

```jsx example
return (
  <Span>
    Наследует&nbsp;
    <Link to='/docs/en/span'>
      Span props
    </Link>
  </Span>
)
```

## Выделяющаяся ссылка

Если ссылка должна выделиться, передайте строку `primary` в свойство `variant`.

```jsx
import { Link, Span } from '@startupjs/ui'
```

```jsx example
return (
  <Row>
    <Span>Inherits&nbsp;</Span>
    <Link block to='/docs/en/span' variant='primary'>
      <Span>Span props</Span>
    </Link>
  </Row>
)
```
