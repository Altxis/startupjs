# Link
Inherits [Span props](/docs/en/typography).

Link allows to navigate of the app and customize anchors.

## Example

```jsx
import { Link, Span } from '@startupjs/ui'
```

```jsx example
return (
  <Span>
    Inherits&nbsp;
    <Link to='/docs/en/typography'>
      Span props
    </Link>
  </Span>
)
```

## Colors

To change link color pass the property `color`.

```jsx
import { Link, Span } from '@startupjs/ui'
```

```jsx example
return (
  <Row>
    <Span>Inherits&nbsp;</Span>
    <Link block to='/docs/en/typography' color='primary'>
      <Span>Span props</Span>
    </Link>
  </Row>
)
```
