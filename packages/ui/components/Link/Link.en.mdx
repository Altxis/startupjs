# Link
Inherits [Span props](/docs/en/Span).

Link allows to navigate of the app and customize anchors.

## Example

```jsx
import { Link, Span } from '@startupjs/ui'
```

```jsx example
return (
  <Span>
    Inherits&nbsp;
    <Link to='/docs/en/span'>
      Span props
    </Link>
  </Span>
)
```

## Stand out Link

If a link needs to stand out pass the `primary` string to `variant` property.

```jsx
import { Link, Span } from '@startupjs/ui'
```

```jsx example
return (
  <Row>
    <Span>Inherits&nbsp;</Span>
    <Link block to='/docs/en/span' variant='primary'>
      <Span>Span props</Span>
    </Link>
  </Row>
)
```
