# Avatar

Avatars can be used to display image, icons, letters.

## Basic Example

```jsx
import { Avatar, Row } from '@startupjs/ui'
```

```jsx example
return (
  <Row>
    <Avatar
      url='/img/avatar1.jpeg'
    />
  </Row>
)
```

## Avatar Status Example

```jsx
import { Avatar, Row } from '@startupjs/ui'
```

```jsx example
return (
  <Row>
    <Avatar
      status='online'
      url='/img/avatar1.jpeg'
    />
    <Avatar
      status='away'
      url='/img/avatar2.jpeg'
    />
  </Row>
)
```

## Avatar Fallback

When `url` prop is not loaded intials of the fallback string are displayed.
In the example below fallback is 'John Doe' therefore 'JD' will be displayed. 
Default `fallback` is `?`.

```jsx
import { Avatar, Row } from '@startupjs/ui'
```

```jsx example
return (
  <Row>
    <Avatar
      url='/img/non-existen-image.jpeg'
    />
    <Avatar
      fallback='John Doe'
      url='/img/non-existen-image.jpeg'
    />
  </Row>
)
```
