# Loader

Loaders are used to show a wait time or a loading state in a page.

## Basic Example

```jsx
import { Loader } from '@startupjs/ui'
```

```jsx example
return (
  <Row align='around'>
    <Loader />
    <Loader duration={2000} />
    <Loader duration={8000} />
  </Row>
)
```

## Loader Size Example

```jsx
import { Loader } from '@startupjs/ui'
```

```jsx example
return (
  <Row align='around'>
    <Loader size='xs' />
    <Loader size='s' />
    <Loader size='m' />
    <Loader size='l' />
    <Loader size='xl' />
    <Loader size='xxl' />
  </Row>
)
```
